How to create a requirements.txt file:
1. Activate your virtual environment, if you haven’t already.<br>
In PowerShell: <br>
`.\venv\Scripts\activate`

2. Generate the requirements.txt file.<br>
In PowerShell:<br>
`pip freeze > requirements.txt`

How to use a requirement.txt file: <br>
In PowerShell:<br>
`pip install -r requirements.txt`
